# Ember Oauth2 Rails

Packaging https://github.com/amkirwan/ember-oauth2 for Rails Asset pipeline for easy inclusion as a gem ;)

## Install

Gemfile

```ruby
gem 'ember-oauth2-rails'
```

bundle!

In js manifest

```
//= require ember.oauth2
```
